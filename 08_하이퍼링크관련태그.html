하이퍼링크 관련 태그 : a(anchor)

        a태그는 현재 HTML문서와 다른 HTML 문서를 연결해서
        페이지 이동을 할 수 있게 하는 기능을 제공.

        -> 이 때, a태그로 감싸진 문자열 == 하이퍼링크 역할

        * 하이퍼링크는 웹(HTML) 문서가 다른 문서와 구분될 수 있는
        핵심적인 기능
        텍스트 or 이미지를 클릭해서 페이지 이동
        * 페이지 내에서 이동 가능
